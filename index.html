GIT FLOW

GIT clone - для того, чтобі "склонить" репозиторий
GIT pull - для того, чтобі получить изменения с репозитория
GIT checkout-b branchname - создать новую ветку и переключится на нее
GIT add. - добавить изменения, которіе сделали локально
GIT commit - m "commit text" - подписать их
GIT push origin branchname - запушить на git сервер
